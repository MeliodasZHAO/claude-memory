# 快速上手指南

5分钟内让记忆伙伴 AI 运行起来！

## 第一步：激活 Skill

在 Claude Code 中说出唤醒词：
- "夏弥"
- "夏弥在吗"
- "夏弥在不在"

系统会自动：
1. 初始化虚拟环境（首次运行）
2. 加载你的记忆数据
3. 以配置的人格回应你

## 第二步：基础交互

### 打招呼
```
你：夏弥在吗？
AI：（探头）在呢，今天想聊啥？
```

### 分享信息
```
你：我最近在学 Python
AI：Python 啊，打算做什么项目？
（系统自动记录这个信息）
```

### 搜索记忆
```
你：我之前记过关于代码的笔记吗？
AI：让我找找...（搜索）找到了XX相关的内容
```

## 第三步：个性化配置

### 修改 AI 名字
```
你：帮我改个名字叫小白
AI：好啊，以后叫我小白就行！
```

### 调整交互风格
AI 会根据你的对话风格自动调整，无需手动配置。

## 常见场景

### 首次见面
如果是第一次使用，AI 会：
- 简单打个招呼
- 在对话中逐步了解你
- 自动建立用户画像

### 日常使用
已经认识后，AI 会：
- 记得之前的对话内容
- 主动关联相关记忆
- 提醒重要日期（如生日）

### 工作记录
```
你：今天处理了 bug，代码优化了性能
AI：（记录中）搞定了？效果怎么样？
```

## 数据位置

所有记忆数据存储在：
```
~/.claude/skills/remembering-anything/user-data/
├── memory/      # 结构化记忆
├── notes/       # 你的笔记
└── config/      # 配置文件
```

## 注意事项

1. **首次运行需要 10-30 秒初始化**（创建虚拟环境）
2. **AI 不会编造记忆**（不知道就会诚实说）
3. **所有对话自动提取记忆**（无需手动保存）
4. **数据完全本地存储**（保护隐私）
